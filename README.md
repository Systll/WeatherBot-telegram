# 🌦️ WeatherBot telegram

Умный Telegram-бот, который говорит правду о погоде, стоит вам только поделиться своим местоположением. Никаких лишних слов — только градусы, осадки и капелька юмора.

---

## 🌟 Основные возможности

*   **📍 Геолокация:** Отправьте боту «точку» на карте, и он мгновенно определит погоду.
*   **🌡️ Точные данные:** Использует API [Open-Meteo](https://open-meteo.com) для получения текущей температуры.
*   **🌧️ Детектор осадков:** Бот подскажет, идет ли сейчас снег 🌨️ или дождь 🌧️.
*   **💬 Живой язык:** Рандомные фразы-подводки делают общение с ботом менее формальным.

---

## 🏗️ Структура проекта

Для чистоты кода мы вынесли настройки в отдельную директорию:

*   `main.py` — основной файл для запуска бота и настройки логирования.
*   `handlers.py` — логика обработки команд и запросов к API.
*   `data/` — папка для конфиденциальных данных (токены, ключи).
*   `requirements.txt` — список необходимых библиотек.
